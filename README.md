# BMI_Calculator_App

This project consist on creating an Android App that calculates BMI ( Body Mass Index )

  * BMI Formula  : weight (kg) / ( height^2 (m))
  * Underweight    : bmi  < 18.5
  * Normal         : 18.4 < bmi < 25.1
  * Overweight     : 25.0 < bmi < 29.9
  * Obesity           : bmi  > 30
